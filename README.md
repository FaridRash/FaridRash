## 🚀 Active Projects

### 🛰️ IGNIS CubeSat — Payload Department *(Supervisor)*
Designing the electro-optical payload for a volcano-monitoring CubeSat using a FLIR Boson 640 microbolometric camera. Work includes:
- Thermal IR radiometric modelling (SNR, NETD, signal chain)
- Optical system simulation in Zemax OpticStudio
- Hazard analysis and payload performance documentation

### 🧠 Brain CT Hemorrhage Segmentation
U-Net segmentation pipeline for intracranial hemorrhage detection in CT scans. Focus on handling severe class imbalance with custom loss functions and data augmentation strategies.

### 📡 Radio Galaxy Morphological Classification
CNN classifier distinguishing FR0, FRI, and FRII radio galaxy morphologies from 50×50 greyscale images. Best model: plain CNN (train ~87%, test ~83%), outperforming ResNet and Optuna-tuned variants. Currently integrating symmetry features post-CNN extraction.

### 📈 Forex RL Trading Agent
DQN-based reinforcement learning agent trading EUR/USD using PyTorch. Custom state representation, stop-loss/target logic (target = 2× SL), epsilon-greedy action selection, and maximum holding period constraints.
